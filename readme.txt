Chinese syllables audiotester
Makarov Edgar(c) 2019

Для работы надо установить python3 (см гугл) и модуль pygame:
1. Запустить командную строку от администратора (см в гугл)
2. pip install pygame
3. Запустить run.bat или "python guess_syllables.py"

Tips:
-Каждый запуск программы - новые слоги
-Чтобы повторить аудио нажмите Enter
-Чтобы выйти наберите ！exit
-Чтобы пропустить слог наберите ！next
